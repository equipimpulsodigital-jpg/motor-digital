Què inclou el manteniment web

El servei de manteniment és una quota mensual que cobreix tot el que necessita una web per funcionar bé de forma contínua. No és suport puntual quan alguna cosa es trenca — és prevenció constant perquè no es trenqui.

  • Còpies de seguretat diàries — si alguna cosa va malament, restaurem la web en qüestió de minuts
  • Actualitzacions de seguretat — pedaços i actualitzacions aplicades de forma controlada per evitar vulnerabilitats
  • Monitorització 24/7 — t'avisem si la web cau abans que ho noti cap client
  • Optimització de rendiment — revisió periòdica de la velocitat i Core Web Vitals
  • Suport tècnic — resolució d'incidències i petits canvis de contingut inclosos
  • Informe mensual — resum de l'estat de la web, visites i incidències resoltes

Per què les pimes de Girona necessiten manteniment web

Moltes empreses creuen que una web es fa i ja no cal tocar-la. La realitat és diferent: les tecnologies evolucionen, apareixen vulnerabilitats de seguretat, els continguts es queden obsolets i el rendiment es degrada amb el temps. Una web abandonada perd posicionament a Google i pot convertir-se en un vector d'atac per a hackers.

El manteniment no és una despesa, és una assegurança. El cost de recuperar una web hackejada o perduda per falta de còpies de seguretat supera amb escreix el cost d'un any de manteniment preventiu.

Hi ha permanència mínima?

No. Pots cancel·lar quan vulguis sense penalització. Confiem que el servei parlarà per ell sol.

Feu manteniment de webs que no heu creat vosaltres?

Sí. Fem una auditoria inicial per valorar l'estat de la web i definir un pla de manteniment adequat. El procés és el mateix tant si la web és nostra com si és d'un altre proveïdor.

Quant triguen a resoldre una incidència?
